Audio Tag

Used to embed sound content in a document.

This tag contains one or more source tags with different video sources. The browser will choose the first source it supports.

Supported audio format:
  1. MP3
  2. WAV
  3. OGV

Attributes
  • autoplay
  • controls
  • height
  • loop
  • muted
  • preload
  • src
